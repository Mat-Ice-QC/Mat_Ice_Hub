
### `values.yaml`

Ce fichier contient toutes les valeurs configurables utilisées dans le chart.

Pour l'instant, le rcon se lie à .10 et non à .190 ; je ne sais pas encore pourquoi.

Champs importants :

``` yaml
image:
  repository: eclipse-temurin
  tag: latest
  pullPolicy: IfNotPresent

minecraft:
  memoryMax: 15G
  memoryMin: 8G
  jarFile: server.jar
  rconPassword: "SuperSecret123"

service:
  type: NodePort
  port: 25565
  nodePort: 32565
  bindIP: 192.168.2.10
  rconBindIP: 192.168.2.190
  rconPort: 25575

persistence:
  hostPath: /mnt/DATA/minecraft
```
