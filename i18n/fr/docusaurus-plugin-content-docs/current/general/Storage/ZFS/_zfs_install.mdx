    
# ZFS : Installation

ZFS est un système de fichiers et gestionnaire de volumes robuste, conçu pour les grandes capacités de stockage, l'intégrité des données et une gestion simplifiée. Ce guide explique comment installer et configurer ZFS sur les distributions Linux courantes.

---


## Prérequis

Avant d'installer ZFS, assurez-vous d'avoir :

* Des privilèges root ou sudo.
* Un système Linux (Ubuntu, Debian, Fedora, CentOS, RHEL, etc.).
* Au moins un périphérique de stockage pour les pools ZFS.

---

## Ubuntu / Debian

1. **Mettez à jour votre liste de paquets :**

```bash
sudo apt update
```

2. **Installez les utilitaires ZFS :**

```bash
sudo apt install zfsutils-linux zfs-dkms -y
```

3. **Chargez le module noyau ZFS :**

```bash
sudo modprobe zfs
```

---

## CentOS / RHEL / Fedora

1. **Activez les dépôts EPEL et ZFS :**

```bash
sudo dnf install -y epel-release
sudo dnf install -y https://zfsonlinux.org/epel/zfs-release.el8_8.noarch.rpm
```

2. **Importez la clé du dépôt ZFS :**

```bash
sudo rpm --import /etc/pki/rpm-gpg/RPM-GPG-KEY-zfsonlinux
```

3. **Installez les paquets ZFS :**

```bash
sudo dnf install -y zfs
```

4. **Chargez le module noyau ZFS :**

```bash
sudo modprobe zfs
```

---

## Vérification de l'installation

Vérifiez que ZFS est correctement installé :

```bash
zfs version
```

Vous devriez voir une sortie similaire à :

```
zfs-2.1.9-1
zfs-kmod-2.1.9-1
```
