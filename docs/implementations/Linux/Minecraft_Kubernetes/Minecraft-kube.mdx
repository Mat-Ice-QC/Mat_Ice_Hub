# Minecraft Kubernetes Helm Chart Guide

## Overview

This document explains all the Helm chart components used to deploy a
Minecraft Java Edition server on Kubernetes with host-mounted storage
and RCON support.

## How to Deploy

### 1. Prepare Host Directory

``` bash
sudo mkdir -p /mnt/DATA/minecraft
sudo chmod -R 777 /mnt/DATA/minecraft
```

### 2. Add your Minecraft server JAR

Place your `server.jar` into:

    /mnt/DATA/minecraft/server.jar


### 3. Install the Helm Chart

``` bash
helm install mc ./minecraft-server -n minecraft --create-namespace
```

### 4. Check Deployment

``` bash
kubectl get pods -n minecraft
kubectl logs -f deployment/minecraft-server -n minecraft
```

### 5. Edit `server.properties`

Example:
``` plaintext
    enable-rcon=true
    rcon.port=25575
    rcon.password=SuperSecret123
    rcon.ip=192.168.2.190
    server-ip=192.168.2.10
    server-port=25565
``` 
Save it to:

    /mnt/DATA/minecraft/server.properties

### 6. Connect to the Server

Use:

    192.168.2.10:25565

### 7. RCON Access

    192.168.2.190:25575
    password: SuperSecret123
